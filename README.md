# VSN: Visual Sales Navigator

**Análisis comercial estratégico impulsado por datos.**

## 🎯 Proyecto
[![Vercel Deployment](https://img.shields.io/badge/Deploy-Vercel-black?logo=vercel)](https://vs-navigator.vercel.app)

VSN es un dashboard analítico de alto impacto diseñado para transformar archivos CSV masivos en una centro de control interactivo. Permite explorar ventas por geografía, tiempo y categorías en un entorno profesional.

## 🚀 Key Features
* 🗺️ **Mapa Coroplético:** Análisis de calor por Estado (US) con escalas matemáticas precisas.
* 📈 **Tendencia Inteligente:** Evolutivo mensual con "Hit Areas" para una interacción táctil fluida.
* 💎 **8 Tarjetas KPI:** Monitoreo en tiempo real de Ventas, AOV (Ticket Promedio) y Segmentación.
* 🔍 **Filtros Cruzados:** Motor de búsqueda de estados y filtros temporales sincronizados.
* 🌓 **Dual Theme:** Cambio instantáneo entre modo reporte y modo analítico.

## 🛠️ Tech Stack
* **Core:** D3.js v7 (Procesamiento de datos y proyecciones).
* **Frontend:** Vanilla JS, HTML5, CSS3 (Variables de entorno).
* **Geodata:** GeoJSON (Fronteras nacionales).

## ⚡ Instalación Rápida
1.  **Clonar:** `git clone https://github.com/jmunozo/vs-navigator.git`
2.  **Servir:** Abrir con `Live Server` (VS Code) o ejecutar `python -m http.server`.
3.  **Explorar:** Navega a `localhost:5500`.

## 📈 Dashboard Insights
* **Finanzas:** Ventas Totales, Ticket Promedio (AOV), Volumen de Órdenes.
* **Estrategia:** Top Categoría, Sub-categoría con mayor rotación y Segmento Predominante.
* **Operaciones:** Modo de envío más frecuente (Logística).
